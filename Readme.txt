Описание проекта:
Разработка модели для предсказания снижения покупательской активности.
Для выполнения работы предоставляются четрые датасета: market_file.csv, market_money.csv, market_time.csv, money.csv.

Для решения задачи бинарной классификации в рамках выполнения проекта использовались модели:
1. KNeighborsClassifier
2. DecisionTreeClassifier
3. SVC
4. LogisticRegression

Подбор лучшей модели и их параметров производился с помощью GridSearchCV.
Лучшей моделью оказалась модель SVC.
